# PrintWise

PrintWise is an engineering assistant for FDM 3D printing.

I am building this project to explore how software can help people make better engineering decisions when 3D printing functional parts.

## Goals

- Analyze 3D models
- Recommend print settings
- Explain engineering reasoning behind recommendations
